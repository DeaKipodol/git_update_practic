 .git 이라는 폴더가 만들어진다. 
 그 폴더에는 
	1)git으로 만든 (관리 해야 할) 버전과
	2)원격저장소의 주소 등 
이 들어있다.

[.git]라는 폴더를 로컬저장소라고 부른다.

git init 명령어는 git을 사용하기위한 초기화 명령어이다.

프로젝트를 시작하고 소스 코드를 Git으로 관리하고자 한다면 먼저 git init으로 저장소를 초기화
